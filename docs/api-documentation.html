OpenCart REST API Documentation

Introduction

This documentation provides information about the OpenCart REST API endpoints and how to use them. The API allows you to interact with various OpenCart features including products, categories, customers, orders, and more.

Base URL

All API URLs referenced in this documentation have the following base:

http://localhost:5000/api

Authentication

Most API endpoints require authentication. The API uses JWT (JSON Web Token) for authentication.

POST /customers/login
Login with customer credentials and receive an authentication token.
curl -X POST http://localhost:5000/api/customers/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "customer@example.com",
    "password": "password123"
  }'
POST /admin/login
Login with admin credentials and receive an authentication token.
curl -X POST http://localhost:5000/api/admin/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "admin123"
  }'
POST /customers/refresh-token
Refresh an expired JWT token.
curl -X POST http://localhost:5000/api/customers/refresh-token \
  -H "Content-Type: application/json" \
  -d '{
    "refresh_token": "your_refresh_token_here"
  }'

Customer

POST /customers/register
Register a new customer account.
curl -X POST http://localhost:5000/api/customers/register \
  -H "Content-Type: application/json" \
  -d '{
    "firstname": "John",
    "lastname": "Doe",
    "email": "john.doe@example.com",
    "telephone": "1234567890",
    "password": "password123"
  }'
GET /customers/profile
Get the current customer's profile information.
curl -X GET http://localhost:5000/api/customers/profile \
  -H "Authorization: Bearer your_token_here"
PUT /customers/profile
Update the current customer's profile information.
curl -X PUT http://localhost:5000/api/customers/profile \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "firstname": "John",
    "lastname": "Smith",
    "telephone": "9876543210"
  }'
POST /customers/change-password
Change the current customer's password.
curl -X POST http://localhost:5000/api/customers/change-password \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "current_password": "password123",
    "new_password": "newpassword123"
  }'
POST /customers/forgot-password
Request a password reset token for a forgotten password.
curl -X POST http://localhost:5000/api/customers/forgot-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com"
  }'
POST /customers/reset-password
Reset a customer's password using a reset token.
curl -X POST http://localhost:5000/api/customers/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com",
    "token": "reset_token_here",
    "newPassword": "newpassword123"
  }'
GET /customers/addresses
Get all addresses for the current customer.
curl -X GET http://localhost:5000/api/customers/addresses \
  -H "Authorization: Bearer your_token_here"
POST /customers/address
Add a new address for the current customer.
curl -X POST http://localhost:5000/api/customers/address \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "firstname": "John",
    "lastname": "Doe",
    "company": "Example Inc.",
    "address_1": "123 Main St",
    "address_2": "Apt 4B",
    "city": "New York",
    "postcode": "10001",
    "country_id": 223,
    "zone_id": 3613
  }'

Product

GET /products
Get a list of products with pagination.
curl -X GET "http://localhost:5000/api/products?page=1&limit=10"
GET /products/:id
Get detailed information about a specific product.
curl -X GET http://localhost:5000/api/products/42
GET /products/purchased
Get a list of products purchased by the current customer.
curl -X GET http://localhost:5000/api/products/purchased \
  -H "Authorization: Bearer your_token_here"

Category

GET /categories
Get a list of all product categories.
curl -X GET http://localhost:5000/api/categories
GET /categories/:id
Get detailed information about a specific category, including its products.
curl -X GET "http://localhost:5000/api/categories/20?page=1&limit=10"

Cart

GET /cart
Get the current customer's shopping cart.
curl -X GET http://localhost:5000/api/cart \
  -H "Authorization: Bearer your_token_here"
POST /cart/add
Add a product to the cart with options.
curl -X POST http://localhost:5000/api/cart/add \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "product_id": 42,
    "quantity": 2,
    "options": [
      {
        "option_id": 1,
        "option_value_id": 3
      }
    ]
  }'
PUT /cart/update
Update the quantity of a cart item.
curl -X PUT http://localhost:5000/api/cart/update \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "item_id": "item_id_here",
    "quantity": 3
  }'
DELETE /cart/remove/:item_id
Remove an item from the cart.
curl -X DELETE http://localhost:5000/api/cart/remove/item_id_here \
  -H "Authorization: Bearer your_token_here"
DELETE /cart/clear
Clear all items from the cart.
curl -X DELETE http://localhost:5000/api/cart/clear \
  -H "Authorization: Bearer your_token_here"

Checkout

POST /checkout/start
Start the checkout process.
curl -X POST http://localhost:5000/api/checkout/start \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json"
POST /checkout/shipping
Add shipping method to checkout.
curl -X POST http://localhost:5000/api/checkout/shipping \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "checkout_id": "checkout_id_here",
    "shipping_method": "Flat Rate",
    "shipping_code": "flat.flat"
  }'
POST /checkout/payment
Add payment method to checkout.
curl -X POST http://localhost:5000/api/checkout/payment \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "checkout_id": "checkout_id_here",
    "payment_method": "Cash On Delivery",
    "payment_code": "cod"
  }'
POST /checkout/complete
Complete the checkout process and create an order.
curl -X POST http://localhost:5000/api/checkout/complete \
  -H "Authorization: Bearer your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "checkout_id": "checkout_id_here",
    "comment": "Please deliver before 6pm."
  }'

Search

GET /search
Search for products with various filters.
curl -X GET "http://localhost:5000/api/search?query=laptop&category=18&price_min=100&price_max=1000&sort=price_asc&page=1&limit=10"
GET /search/filters
Get available search filters (categories, price ranges, etc.).
curl -X GET http://localhost:5000/api/search/filters
